# CLI Reference

**Command-line interface technical specifications.**

---

## Purpose

This directory contains **information-oriented** technical reference documentation for ragged's command-line interface. These documents provide precise specifications for all CLI commands, options, and exit codes.

**Use this documentation when you need to:**
- Look up exact command syntax
- Find available options and their defaults
- Check exit codes and error conditions
- Verify environment variable names
- Reference technical specifications

**Don't use this for:**
- Learning how to use ragged (see [CLI Features Guide](../../guides/cli/cli-features.md))
- Step-by-step tutorials (see [tutorials](../../tutorials/))
- Understanding concepts (see [explanation](../../explanation/))

## Style

CLI reference documentation is:
- **Dry and precise** - Technical specifications only
- **Complete** - Every option documented
- **Consistent** - Same format for all commands
- **Concise** - Minimal examples, no narrative
