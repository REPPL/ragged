# Design Documentation

This directory contains design documentation for ragged's user interfaces and visual elements.

## Purpose

Design specifications, mockups, wireframes, and visual assets for ragged's interfaces.

## Contents

- **[webUI/](./webUI/)** - Web user interface design documentation and assets

## What Belongs Here

- UI/UX design specifications
- Visual design systems and style guides
- Wireframes and mockups
- Interface design files (.svg, .penpot, etc.)
- Icon sets and visual assets
- User flow diagrams
- Accessibility design considerations

## What Doesn't Belong Here

- **Implementation code** → See `src/` directory
- **Architecture decisions** → See `docs/development/decisions/adrs/`
- **User guides** → See `docs/guides/`
- **API specifications** → See `docs/reference/`
