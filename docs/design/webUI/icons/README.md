# Web UI Icons

Icon assets and specifications for ragged's web user interface.

## Purpose

This directory contains icon design files, exported assets, and icon design specifications for the ragged web UI.

## Contents

Icon design files in various formats:
- SVG source files
- Penpot design files
- Exported icon sets

## What Belongs Here

- Icon design source files (.svg, .penpot)
- Icon sets and sprite sheets
- Icon usage specifications
- Accessibility considerations for icons
- Icon colour and sizing standards

## What Doesn't Belong Here

- **Wireframes** → See `../wireframe/`
- **Full UI mockups** → See `../`
- **Implementation code** → See `src/web/`

## Icon Design Principles

When designing icons for ragged:
1. Maintain consistency with existing icon style
2. Ensure clarity at small sizes (16x16, 24x24)
3. Support dark/light/high-contrast themes
4. Include accessible text alternatives
5. Use semantic naming conventions
