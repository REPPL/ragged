# CLI Guides

**Task-oriented guides for using ragged's command-line interface.**

---

## Purpose

This directory contains **task-oriented** guides that help you accomplish specific goals using ragged's CLI. These guides assume basic familiarity and focus on practical workflows and best practices.

**Use these guides when you want to:**
- Learn how to use CLI features effectively
- Accomplish specific tasks step by step
- Understand practical workflows
- Follow best practices
- Troubleshoot common issues

**Don't use these guides for:**
- Quick syntax lookups (see [CLI Reference](../../reference/cli/command-reference.md))
- Learning ragged from scratch (see [tutorials](../../tutorials/))
- Understanding internal architecture (see [explanation](../../explanation/))

## Guide Structure

Each guide follows this pattern:
1. **Goal** - What you'll accomplish
2. **Prerequisites** - What you need
3. **Steps** - Clear instructions with examples
4. **Expected Outcome** - Verification
5. **Troubleshooting** - Common issues
6. **Best Practices** - Tips and recommendations
7. **Next Steps** - Related guides
