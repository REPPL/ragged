**Learning-oriented** - Practical lessons that help you build foundational skills.

## What Are Tutorials?

Tutorials are **learning-oriented** guides that take you through hands-on exercises step-by-step. They're designed for beginners who want to learn by doing.

**Good for**:
- New users with no prior RAG experience
- Learning ragged basics through practical examples
- Building confidence through safe-to-fail exercises

**Not good for**:
- Solving specific problems (see [guides](../guides/))
- Looking up technical details (see [reference](../reference/))
- Understanding concepts deeply (see [explanation](../explanation/))

## Tutorial Structure

Each tutorial includes:
1. **Learning Objectives** - What you'll learn
2. **Prerequisites** - What you need before starting
3. **Step-by-Step Instructions** - Detailed, beginner-friendly steps
4. **Code Examples** - Complete, working code you can copy
5. **What's Next** - Where to go after completing the tutorial
