## Time Log Format

Each time log includes:
- **Total hours** per category (implementation, testing, debugging, docs)
- **AI-assisted vs. manual** time breakdown
- **AI effectiveness** by task type
- **Challenges** encountered and resolutions
- **Learning** and retrospective insights
