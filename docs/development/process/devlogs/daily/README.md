# Daily Development Logs

**Purpose:** Day-by-day development narratives

**Last Updated:** 2025-11-15

---

## Overview

This directory contains daily development logs organised by date.

**Format:** `YYYY-MM-DD.md`

**Contents:** Daily progress, challenges, decisions, and learnings

---

## Purpose

Daily logs provide a chronological narrative of development progress, complementing version-based development logs.

**Use cases:**
- Track daily progress
- Record spontaneous decisions
- Document blockers and solutions
- Capture learnings and insights

---

## For Contributors

Create a daily log when:
- Working on ragged development
- Making significant progress or decisions
- Encountering interesting challenges
- Learning something worth documenting

**Template:** See `../../templates/daily-log-template.md`
