# v0.6.10: Public API Launch

**Focus:** External developer API access with SDKs

**Hours:** 10-15 hours

**Dependencies:** v0.6.6 (FastAPI stable), v0.6.7-v0.6.9 (Svelte UI)

**Status:** Planned

---

## Overview

Version 0.6.10 transforms the internal FastAPI (v0.6.6) into a public developer API with versioning, API key management, client SDKs, and external documentation.

**What This Version Delivers:**
- API versioning (`/v1/...`)
- API key management and authentication
- Rate limiting with tiered quotas
- Client SDKs (Python, JavaScript/TypeScript)
- Developer documentation portal

**Why After UI:**
- API proven stable through internal use (v0.6.7-v0.6.9)
- Allows time for API design refinement
- Svelte UI validates API completeness

---

## Key Features

### API Infrastructure
- **Versioning:** `/v1/...` prefix (allows future `/v2/...`)
- **Authentication:** API key generation and rotation
- **Rate Limiting:** Per-key quotas (free, developer, enterprise tiers)
- **Documentation:** Developer portal with examples

### Client SDKs

**Python SDK (`ragged-client`):**
```python
from ragged_client import RaggedClient

client = RaggedClient(api_key="rg_...")
results = client.query.text("neural architecture")
```

**JavaScript SDK (`@ragged/client`):**
```typescript
import { RaggedClient } from '@ragged/client';

const client = new RaggedClient({ apiKey: 'rg_...' });
const results = await client.query.text('neural architecture');
```

## Rate Limiting Tiers

**Free Tier:**
- 1,000 requests/month
- 10 documents/month
- Community support

**Developer Tier ($29/month):**
- 50,000 requests/month
- 100 documents/month
- Email support

**Enterprise Tier (Custom):**
- Unlimited requests
- Unlimited documents
- Dedicated support
