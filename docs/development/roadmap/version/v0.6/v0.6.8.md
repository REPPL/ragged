# v0.6.8: Svelte Advanced Features

**Focus:** Visualisations, monitoring dashboards, and advanced document preview

**Hours:** 25-35 hours

**Dependencies:** v0.6.7 (Svelte Core UI)

**Status:** Planned

---

## Overview

Version 0.6.8 adds advanced visualisations and monitoring capabilities to the Svelte UI. This version implements the interactive knowledge graph, GPU monitoring dashboard, enhanced document preview with highlighting, and real-time performance analytics.

**What This Version Delivers:**
- Interactive knowledge graph visualisation (D3.js)
- Real-time GPU monitoring dashboard
- Document preview with highlighting and page thumbnails
- Storage analytics visualisations
- Query performance metrics
- WebSockets for real-time updates

---

## Key Features

### Knowledge Graph Visualisation (D3.js)
- Interactive force-directed graph
- Node types: documents, entities, topics
- Zoom, pan, and filter controls
- Click to explore related documents
- Dynamic layout algorithms

### GPU Monitoring Dashboard
- Real-time GPU utilisation charts
- Memory usage tracking
- Temperature monitoring
- Model loading status
- Historical performance trends

### Enhanced Document Preview
- PDF page thumbnails
- Text highlighting in preview
- Matched chunk location indicators
- Page-by-page navigation
- Full-screen preview mode

### Analytics Visualisations
- Storage usage pie charts
- Query latency trends
- Cache effectiveness graphs
- Domain distribution charts
- Model routing statistics
