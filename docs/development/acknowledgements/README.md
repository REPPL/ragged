# Acknowledgements

ragged builds upon innovative work from the open-source community, research papers, and individual developers. This directory documents projects and research that have influenced ragged's design and implementation.

---

## Projects

**[Autonomous Agentic RAG](./autonomous-agentic-rag.md)** by Fareed Khan
- Self-improving RAG systems with evolutionary optimisation
- Influenced: v0.5.x (model routing), v1.x (multi-agent), v2.x (compliance)

**[LEANN](./leann.md)** - Graph-Based Vector Storage
- 97% storage savings through selective recomputation
- Influenced: v0.4.9 (LEANN backend integration)

---

## Research Papers

**[Context Engineering 2.0](./context-engineering-2.0.md)** by GAIR
- Theoretical framework for entropy reduction through structured context
- Influenced: v0.4.x (knowledge graphs), v0.6.0 (context adaptation)

See individual files for detailed academic citations.
