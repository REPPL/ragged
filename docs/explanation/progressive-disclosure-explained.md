### What is Progressive Disclosure?
- The UX principle explained
- Show only what users need, when they need it
- Reduce cognitive load
- Support both beginners and experts

### Application to ragged

#### Basic Usage
- Simple chat interface
- Upload and ask questions
- Minimal configuration
- Sensible defaults

#### Intermediate Features
- Document collections
- Search settings
- Response formatting
- History and bookmarks

#### Advanced Capabilities
- Model selection
- Retrieval tuning
- Evaluation metrics
- Developer mode
- API access

### Interface Design
- Layered complexity
- Contextual help
- Gradual feature introduction
- Expert shortcuts

### Examples in ragged

#### Query Interface
- **Basic**: Simple text box
- **Intermediate**: Filters, collection selection
- **Advanced**: Query DSL, retrieval parameters

#### Results Display
- **Basic**: Answer with sources
- **Intermediate**: Confidence scores, alternative answers
- **Advanced**: Retrieval metrics, chunk inspection

#### Configuration
- **Basic**: Auto-detected settings
- **Intermediate**: Model selection, performance tuning
- **Advanced**: Full configuration control

### Benefits
- Faster onboarding
- Reduced overwhelm
- Scalable learning
- Expert efficiency

### Trade-offs
- Discoverability challenges
- Design complexity
- Documentation requirements
- Testing burden
